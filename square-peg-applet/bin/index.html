Square Peg Applet


Conjecture [Square Peg Problem, Toepliz 1911]: Every continuous simple closed curve in the plane contains the four points of a square.

The Square Peg Problem is only know to hold true for curves that are "smooth enough", e. g. C1 does it. Many proofs for different curve classes are known.

Conjecture [Rectangular Peg Problem]: Every smooth simple closed curve in the plane contains the four points of a rectangle with a prescribed aspect ratio r:1.

The Rectangular Peg Problem (for smooth curves) is only known to hold true for r=1, that is, for squares.
